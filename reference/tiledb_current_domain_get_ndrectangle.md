# Get a `tiledb_ndrectangle` from a `tiledb_current_domain` object

Get a `tiledb_ndrectangle` from a `tiledb_current_domain` object

## Usage

``` r
tiledb_current_domain_get_ndrectangle(cd)
```

## Arguments

- cd:

  A TileDB CurrentDomain object

## Value

The corresponding TileDB NDRectangle object
