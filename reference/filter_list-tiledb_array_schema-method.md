# Returns the offsets and coordinate filter_lists associated with the `tiledb_array_schema`

Returns the offsets and coordinate filter_lists associated with the
`tiledb_array_schema`

## Usage

``` r
# S4 method for class 'tiledb_array_schema'
filter_list(object)
```

## Arguments

- object:

  A TileDB Schema object

## Value

A list of `tiledb_filter_list` objects
