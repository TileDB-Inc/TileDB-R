# Sets the TileDB Filter List for the TileDB Attribute object

Sets the TileDB Filter List for the TileDB Attribute object

## Usage

``` r
# S4 method for class 'tiledb_attr'
filter_list(x) <- value
```

## Arguments

- x:

  TileDB Attribute

- value:

  TileDB Filter List

## Value

The modified TileDB Attribute object
