# Returns the TileDB Filter List object associated with the given TileDB Dimension

Returns the TileDB Filter List object associated with the given TileDB
Dimension

## Usage

``` r
# S4 method for class 'tiledb_dim'
filter_list(object)
```

## Arguments

- object:

  A `tiledb_dim` object

## Value

A `tiledb_filter_list` object
