# Retrieve query_condition value for the array

A `tiledb_array` object can have a corresponding query condition object.
This methods returns it.

## Usage

``` r
query_condition(object)

# S4 method for class 'tiledb_array'
query_condition(object)
```

## Arguments

- object:

  A `tiledb_array` object

## Value

A `tiledb_query_condition` object
