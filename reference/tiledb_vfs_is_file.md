# Test for VFS File

Test for VFS File

## Usage

``` r
tiledb_vfs_is_file(uri, vfs = tiledb_get_vfs())
```

## Arguments

- uri:

  Character variable with a URI describing a file path

- vfs:

  A TileDB VFS object; default is to use a cached value.

## Value

A boolean value indicating if it is a file
