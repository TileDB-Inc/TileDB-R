# Creates a 'tiledb_query_condition' object

Creates a 'tiledb_query_condition' object

## Usage

``` r
tiledb_query_condition(ctx = tiledb_get_context())
```

## Arguments

- ctx:

  (optional) A TileDB Ctx object; if not supplied the default context
  object is retrieved

## Value

A 'tiledb_query_condition' object
