# Load a saved 'tiledb_profile' object

Load a saved 'tiledb_profile' object

## Usage

``` r
tiledb_profile_load(name = NULL, dir = NULL)
```

## Arguments

- name:

  (optional) Name of the profile to load

- dir:

  (optional) Directory where the profile to load is saved

## Value

The loaded 'tiledb_profile' object
