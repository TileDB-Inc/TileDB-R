# Return a TileDB Group URI

Return a TileDB Group URI

## Usage

``` r
tiledb_group_uri(grp)
```

## Arguments

- grp:

  A TileDB Group object as for example returned by
  [`tiledb_group()`](https://tiledb-inc.github.io/TileDB-R/reference/tiledb_group.md)

## Value

A character value with the URI
