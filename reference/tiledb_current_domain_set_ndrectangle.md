# Set a `tiledb_ndrectangle` in a `tiledb_current_domain` object

Set a `tiledb_ndrectangle` in a `tiledb_current_domain` object

## Usage

``` r
tiledb_current_domain_set_ndrectangle(cd, ndr)
```

## Arguments

- cd:

  A TileDB CurrentDomain object

- ndr:

  A TileDB NDRectangle object

## Value

The modifiled TileDB CurrendDomain object
