# tiledb - Interface to the TileDB Storage Manager API

The efficient multi-dimensional array management system 'TileDB'
introduces a novel on-disk format that can effectively store reads. It
features excellent compression, an efficient parallel I/O system which
also scales well, and bindings to multiple languages.
