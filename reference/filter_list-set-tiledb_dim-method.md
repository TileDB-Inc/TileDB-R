# Sets the TileDB Filter List for the TileDB Dimension object

Sets the TileDB Filter List for the TileDB Dimension object

## Usage

``` r
# S4 method for class 'tiledb_dim'
filter_list(x) <- value
```

## Arguments

- x:

  A `tiledb_dim` object

- value:

  TileDB Filter List

## Value

The modified TileDB Dimension object
